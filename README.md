# 2D-Arrays

A module focused on mastering 2D Arrays in Java. 
Covering matrix input/output, searching, and 
advanced traversal algorithms.

## 🚀 Programs Implemented

- ✅ **Matrix_Transpose.java**: Implementation of Matrix Transpose.
- ✅ **Matrix_Search.java**: Search for a given element x in a matrix and print its indices.
- ✅ **Spiral_Traversal.java**: Print matrix elements in Spiral Order using boundary traversal.

## 🛠️ Concepts Covered

- 2D Array Declaration & Initialization
- Nested Loop Traversal (row × column)
- Matrix Transpose Logic
- Linear Search in 2D Arrays
- Spiral Traversal — O(n×m)

## 📊 Time Complexity

| Operation | Complexity |
|-----------|------------|
| Traversal | O(n × m) |
| Search | O(n × m) |
| Transpose | O(n × m) |
| Spiral Order | O(n × m) |
